1. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web?

CSS dan JavaScript digunakan untuk meningkatkan interaktivitas dan estetika dari website.

2. Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web?

CSS dan JavaScript dapat diintegrasikan melalui tag <link> dan <script> di dalam HTML.

3. Jelaskan apa saja keuntungan menggunakan JavaScipt dan CSS?

Keuntungan menggunakan JavaScript dan CSS meliputi kecepatan, fleksibilitas, dan kemudahan dalam mengubah tampilan web.

4. Jelaskan apa saja keuntungan menggunakan Bootstrap?

Bootstrap menyediakan kerangka kerja responsif yang memudahkan pengembangan UI.

5. Berikan salah satu contoh penggunaan Bootstrap lalu jelaskan fungsinya serta berikan screenshot hasil running di web-nya!

Contoh penggunaan Bootstrap adalah penggunaan navbar, yang memudahkan navigasi pengguna.
