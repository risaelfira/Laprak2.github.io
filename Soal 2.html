Bagaimana CSS Dan JavaScript Dapat Diintegrasikan Dalam Pemrograman Web?

Mengintegrasi CSS :

  • CSS Inline

    Gaya diterapkan langsung ke dalam elemen HTML individual menggunakan atribut style. Metode ini digunakan guna menerapkan gaya unik ke elemen tertentu. Namun, tidak disarankan untuk menata sebagian besar situs web karena masalah pemeliharaan.

  • CSS Internal

    Metode ini diterapkan dengan menyematkannya di dalam tag <style> di bagian <head> dokumen HTML. Metode ini cocok digunakan saat menata suatu halaman karena gaya tidak dapat digunakan kembali di seluruh halaman.

    Contoh :

  • CSS Eksternal

    Metode ini mwnghubungkan file .css terpisah menggunakan tag <link> di bagian <head> dokumen HTML. Metode ini yang paling umum digunakan dan direkomendasikan untuk menata situs web, terutama yang lebih besar. Metode ini memungkinkan gaya yang konsisten di seluruh halaman dan pemeliharaan yang lebih mudah. File CSS eksternal dapat di-cache oleh browser, meingkatkan waktu pembuatan halaman. Sintaks dari tag <link> meliputi atribut rel yang diatur ke “stylesheet” dan atribut href yang menunjuk ke jalus CSS.

Mengintegrasikan JavaScript :

  • JavaScript Internal

    Kode JavaScript disematkan langsung di dalam tag <script> dalam dokumen HTML yang ditempatkan pada bagian <head> atau <body> tergantung kapan skrip akan dieksekusi. Metode ini cocok digunakan untuk kode dalam jumlah kecil.

    Contoh :

  • JavaScript Eksternal

    Metode ini menghubungkan file .js secara terpisah menggunakan tag <script> dengan atribut src. Atribut src menentukan jalur ke file JavaScript eksternal. Metode ini sering dipakai untuk proyek yang lebih besar karena meningkatkan organisasi dan penggunaan kembalikode yang lebih baik. Tag <script> untuk file eksternal biasanya ditempatkan pada akhir <body> atau dapat menggunakan async atau defer untuk mengontrol pemuatan dan eksekusi skrip.

    Contoh :

  • JavaScript Inline

    Metode ini menggunakan atribut event handler (misalnya onclick, onmouseover) langsung di dalam elemen HTML untuk mengeksekusi kode JavaScript. Metode ini umumnya tidak disarankan karena pemeliharaan dan pemisahan tanggung jawab yang buruk.

    Contoh :

    <button onclick="console.log('Tombol dihover!')">Hover Saya</button>
